## ⚙️ Cómo funciona

1. Cliente llena el formulario de 5 pasos
2. Sistema lo clasifica automáticamente (5 segmentos)
3. Cliente abre WhatsApp con mensaje pre-llenado
4. Admin ve el descuento en el panel y cotiza

## 📊 Segmentos y descuentos

| Segmento | Descuento |
|----------|-----------|
| Cliente Final | 0% |
| Sodas / Rest. Pequeños | 12% |
| Restaurantes Medianos | 20% |
| Hoteles / Catering | 25% |
| Mayorista | 32% |
